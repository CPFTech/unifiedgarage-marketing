Changelog.

A running list of what we've shipped. We push small improvements daily; this page captures the ones worth knowing about.

May 2026

QuickFlip-inspired sweep

Added

  • Showroom TV mode at /p/[slug]/floor — public auto-refresh dashboard for wall-mounted screens
  • Window stickers via the print library — single-click branded window flyer with hero pricing
  • Vehicle photo categorization — Cover/Exterior/Interior/Damage/Wheels/Detail tabs on inventory detail
  • Customer reservation hold — sales reps can hold a vehicle for a named customer with expiry
  • Retail-ready approval workflow — manager sign-off before a vehicle becomes active inventory, with audit trail
  • Public trade-in appraisal form — customers submit at /p/[slug]/trade; lands in the appraisal queue
  • User custom title chips — "Sales mgr", "Used car mgr" alongside role
  • Trial-end banner — running countdown for dealerships in trial
April 2026

Tier 1 reports + customer MPI

Added

  • Daily Operating Control (DOC) dashboard — service today, deals MTD/YTD, WIP at a glance
  • Technician productivity report — hours sold, RO + line counts per tech
  • Advisor productivity report — ROs written, hours/RO, parts$/RO, customer-pay mix
  • Multi-point inspection (MPI) with customer-facing approval link — photos + per-item Approve/Decline via SMS or email
  • Declined-work follow-up queue — auto-tracks customer-declined recommendations for re-marketing
  • Print library — shared infrastructure for RO, deal jacket, PO, inspection report, daily close
  • Print settings persistence — per-doc paper/margin preferences sync across devices
March 2026

App redesign + auth hardening

Added

  • Direction C aesthetic — light zinc canvas, amber accents, editorial sidebar across every page
  • PageHeader / StatRow components — unified visual rhythm
  • CSRF protection on every mutating endpoint
  • TOTP MFA with recovery codes
  • Failed-login lockout with exponential backoff
  • Editorial sidebar nav with role-filtered links
February 2026

RO intake + service polish

Added

  • PBS/Tekion-style RO intake — fuel level, key tag, appointment source, transportation type, insurance/warranty claim, deductible
  • Technician view — desktop queue with line-level punch, hold-parts/hold-tech/hold-auth states
  • Service appointments calendar with capacity awareness
