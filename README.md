# unifiedgarage-marketing

Marketing website for **UnifiedGarage** — served at `https://unifiedgarage.com` and `https://www.unifiedgarage.com`.

This is a single-page static site. The HTML is self-contained (inline CSS, Inter font loaded from Google Fonts). No build step.

## Deploy

Hosted on Vercel. Pushes to `main` auto-deploy.

## Related projects

- `my-dms` — NestJS backend API at `https://api.unifiedgarage.com`
- `my-dms-frontend` — DMS frontend at `https://app.unifiedgarage.com`
